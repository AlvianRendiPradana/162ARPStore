		  Store = adalah nama yang digunakan untuk usaha kecil dan besar

Cara menjalankan website :
1. Buka folder yang sudah saya kirimkan TOKO ONLINE-ALVIAN RENDI PRADANA-24083000162-2G
2. Buka folder pages
3. Lalu pilih file yang sudah saya namakan dengan homepage.html
4. Lanjutkan untuk membuka file tersebut bisa dibuka dengan Google Chrome/Microsoft Edge
5. Anda akan ditampilkan dengan tampilan pertama yaitu home/beranda toko online saya
6. Jika anda belum memiliki akun, anda bisa memilih fitur DAFTAR
7. Lalu ketika anda sudah di halaman fitur DAFTAR, anda akan dimintai data pendaftar
8. Jika sudah daftar, anda bisa langsung ke halaman MASUK
9. Berikutnya anda bisa memasukan data yang sesuai, saat di halaman DAFTAR sebelumnya
10. Jika sudah, anda bisa melihat berbagai produk yang ada di website toko online saya
11. Disitu juga sudah saya sediakan fitur halaman KERANJANG, anda bisa melihatnya/pilih halaman tersebut
12. Jika anda sudah dihalaman KERANJANG tersebut, anda bisa melihat produk apa saja yang akan di checkout/beli

Cara melihat single product :
1. Buka folder yang sudah saya kirimkan TOKO ONLINE-ALVIAN RENDI PRADANA-24083000162-2G
2. Buka folder pages
3. Lalu pilih file yang sudah saya namakan dengan singleproduct.html
4. Lanjutkan untuk membuka file tersebut bisa dibuka dengan Google Chrome/Microsoft Edge
5. Anda akan ditampilkan dengan tampilan single product dari website toko online saya
6. Lalu disitu mendeskripsikan tentang produk dan adanya Stok Produk tersebut, dan juga ada deskripsi tentang Review Produk dari konsumen

Fitur-fitur yang ditambahkan : - Fitur Beranda
			       - Fitur Notifikasi
			       - Fitur Keranjang
			       - Fitur Daftar
			       - Fitur Masuk/Login
